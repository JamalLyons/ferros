# ferros-ui

Optional TUI/GUI for Ferros debugger visualization.

## Overview

`ferros-ui` provides user interface components for visualizing and interacting with the Ferros debugger, including:

- Terminal User Interface (TUI) using `ratatui`
- Graphical User Interface (GUI) using `egui` (optional)
- Source code view
- Stack frame visualization
- Local variable inspection
- Heap visualization

## Usage

Add `ferros-ui` to your `Cargo.toml`:

```toml
[dependencies]
ferros-ui = "0.0.0"
```

## Features

- Terminal-based interface for debugging sessions
- Graphical interface option
- Real-time visualization of debug state
- Interactive debugging controls
